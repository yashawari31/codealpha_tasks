# 🏦 Credit Scoring Prediction

A Machine Learning project that predicts whether a customer's credit score is **High, Average, or Low** using demographic and financial information.

## 🚀 Features

- Predicts credit score category
- Interactive Streamlit web app
- Decision Tree Classifier
- User-friendly interface

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 🤖 Model

- Decision Tree Classifier
- Accuracy: **90.48%**
- Cross Validation Accuracy: **93.10%**

## 📂 Files

- `app.py` – Streamlit application
- `credit_scoring_model.pkl` – Trained model
- `label_encoders.pkl` – Encoders
- `Credit_scoring.ipynb` – Model training notebook
- `requirements.txt` – Dependencies

## ▶️ Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
